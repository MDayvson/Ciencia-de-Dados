---

## 🚀 Tecnologias e Ferramentas

No decorrer do curso, utilizei o ecossistema de dados do Python para resolver problemas de negócio reais:

- **Análise e Manipulação:** `Pandas`, `NumPy`
- **Visualização:** `Plotly`, `Seaborn`, `Matplotlib`
- **Machine Learning:** `Scikit-Learn` (Regressão, Classificação, Clustering)
- **Ambiente:** `Jupyter Notebook`, `Anaconda`, `Google Colab`

## 💡 Competências Desenvolvidas

- **Tratamento de Dados (ETL):** Limpeza, transformação e preparação de grandes bases de dados.
- **Análise Exploratória (EDA):** Extração de insights através de correlações e estatística descritiva.
- **Modelagem Preditiva:** Treinamento, validação e ajuste de hiperparâmetros de modelos.
- **Data Storytelling:** Comunicação de resultados técnicos de forma clara para áreas de negócio.
